post(self)

source code 

Respuesta POST que edita los campos del usuario pasados como parámetro. El identificador del usuario es su nombre, y este campo no será modificable.

En caso de que el usuario que intente acceder a este método no sea del grupo "Admin" recibirá el mensaje: "Unauthorized".

Parameters:
  • edit-full_name (str) - nombre completo nuevo
  • edit-email (str) - email nuevo
  • edit-group (str) - grupo nuevo
  • name (str) - identificador de usuario que se va a modificar
