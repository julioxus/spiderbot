getAllLinks(root, depth, max_pages, onlyDomain, reg='.*')

source code 

Algoritmo principal de la araña. Obtiene los links de la página raíz root con una búsqueda en anchura, determinada por una profundidad de búsqueda depth y número máximo de páginas max_pages. Puede filtrar páginas si se indica que sólo busque en su propio dominio o usar una expresión reguar reg.

Parameters:
  • root (str) - página raíz
  • depth (int) - profundidad de búsqueda
  • max_pages (int) - links que se obtendrán como máximo
  • onlyDomain (bool) - indica si se quieren filtar páginas que estén sólo en su dominio
  • reg (str) - expresión regular para filtrar páginas en la obtención de links
Returns: list
lista de enlaces compuesta por objetos de tipo tupla con dos valores:
  • 0: nombre del enlace
  • 1: tipo de enlace: html o css
